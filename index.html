Praktikum PDW 2024

CSS merupakan singkatan dari Cascading Style Sheets. Ini adalah bahasa yang digunakan untuk mengendalikan tampilan dan format dari elemen-elemen HTML di halaman web. Dengan menggunakan CSS, Anda dapat mengubah warna, ukuran, jenis huruf, letak, dan banyak aspek lain dari elemen-elemen HTML.
